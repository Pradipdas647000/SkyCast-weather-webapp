# **App Name**: ForecastAI

## Core Features:

- Current Weather Display: Display current weather conditions, including temperature, humidity, wind speed, and AQI.
- 7-Day Forecast: Show the 7-day weather forecast.
- Hourly Forecast: Present an hourly weather forecast with an interactive chart (Recharts or Chart.js).
- City Search: Enable users to search for cities with an autocomplete feature.
- Geolocation Detection: Detect the user's location to provide weather information.
- User Authentication: Allow users to authenticate via Email/Password and Google Sign-in.
- AI Weather Summary: Use Cloud Functions to securely call the OpenWeatherMap API and generate an AI-based summary of the weather conditions with a tool that helps decide what information to include based on relevance to the user.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to evoke a sense of trust and reliability, reflecting the importance of accurate weather forecasts.
- Background color: Light gray (#F0F4F8) to ensure readability and a modern, clean aesthetic.
- Accent color: Bright cyan (#4DD0E1) to highlight interactive elements and CTAs, adding a touch of vibrancy.
- Font: 'Inter' for both headlines and body text, providing a modern and neutral aesthetic. This sans-serif font ensures readability and clarity across the app.
- Use clear and recognizable weather icons (e.g., rain, snow, clouds) for easy interpretation.
- Modern, responsive UI with a mobile-first approach, ensuring seamless experience across devices.
- Smooth animations for theme toggling and dynamic weather backgrounds.
